# Global-Travel-Insights
Project Overview

This project analyzes logistics and shipment operations to evaluate transportation efficiency, carrier performance, delivery reliability, and operational costs. The objective is to transform shipment data into actionable business insights that support logistics optimization and data-driven decision-making.

The solution was developed using Power BI and focuses on key logistics KPIs, shipment performance monitoring, and operational efficiency analysis.

Business Objectives
Monitor shipment performance across carriers and destinations.
Evaluate delivery success and operational reliability.
Analyze transportation costs and cost efficiency.
Measure transit performance and delivery speed.
Identify operational bottlenecks and improvement opportunities.
Support logistics decision-making through interactive dashboards.
